# Authentication & Authorization Plan

## Approach: ASP.NET Core Identity + JWT Bearer

### Key Decisions
- **Separate Identity DbContext** - Don't mix Identity tables with business tables
- **Infrastructure layer** owns Identity setup (framework concern)
- **Application layer** owns Auth CQRS (follows existing pattern)
- **JWT Access Token** (15 min) + **Refresh Token** (7 days)
- **3 Roles**: Admin, Officer, Customer
- **Domain layer untouched** (zero dependencies rule)

## Step 5: Protect Existing Endpoints

Add `[Authorize]` attributes to all controllers:

```csharp
[ApiController]
[Route("api/[controller]")]
[Authorize]  // <-- All endpoints require auth by default
public class AccountsController : ControllerBase
{
    // Admin-only endpoints use [Authorize(Roles = "Admin")]
    // Teller endpoints use [Authorize(Roles = "Admin,Officer")]
}
```
